# Harmonix - Dynamic Web Audio Player 🎵

A sleek, responsive web-based audio player built with Vanilla JavaScript. Features dynamic track rendering, real-time search filtering, and custom media controls for a seamless music listening experience.

## ✨ Features
* **Custom Audio Engine:** Full control over play, pause, and track progression using the HTML5 Audio API.
* **Real-Time Search:** Instantly query the music library with zero latency; dynamically updates the UI based on user input.
* **Data-Driven UI:** Album cards and track lists are generated dynamically via JavaScript arrays, making the application highly scalable.
* **Error Handling:** Elegant, self-dismissing UI alerts for missing tracks or invalid searches.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3
* **Logic & Functionality:** Vanilla JavaScript (ES6+)
